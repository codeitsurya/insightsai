## Setup - Installation & Running App

### Running Backend app


Install python latest version
update PATH with 
C:\Users\Admin\AppData\Local\Programs\Python\Python314\Scripts\C:\Users\Admin\AppData\Local\Programs\Python\Python314\C:\Users\Admin\AppData\Local\Programs\Python\Launcher\;

```bash
  cd backend
  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements.txt
  uvicorn main:app --reload
```
http://127.0.0.1:8000 - App running

### Running Frontend App
```
npm install
npm run dev
```
http://localhost:3000
